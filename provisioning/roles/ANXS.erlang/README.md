## ANXS - erlang [![Build Status](https://travis-ci.org/ANXS/erlang.png)](https://travis-ci.org/ANXS/erlang)

Ansible role which installs Erlang


#### Requirements & Dependencies
- Tested on Ansible 1.4 or higher.

erlang_build_flags: ""
